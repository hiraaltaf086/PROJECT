80% of all blindness can be prevented

And the first step is early and accessible screening. Eyesion is a screening software, which helps eye care professionals perform screenings for eye diseases in a safe and efficient manner.

Machine Learning Algorithms have been used to detects signs of diabatic retinopathy (condition of pathalogical myopia) by analyzing fundus images. It can be used as a clinical decision support system (CDSS)

It works in 3 easy steps:

The image is transferred automatically to the Eyesion online software.

Upload the image to Eyesion

Image Processing is performed including: Image Enhancement, Image Segmentation, Feature Extraction

Results will be displayed if any abnormalities are found
